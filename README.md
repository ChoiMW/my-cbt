# 웹 기반 CBT(Computer-Based Training) 시스템

웹 브라우저 환경에서 간단하게 JSON 형식의 문제 데이터를 업로드하여 시험을 응시하고, 오답 노트까지 관리할 수 있는 가벼운 CBT 시스템입니다.

## 주요 기능 (Features)

1. **손쉬운 시험 등록**
   - 별도의 서버나 데이터베이스 없이 JSON 파일을 첨부하거나 직접 텍스트를 붙여넣어 즉시 시험을 생성할 수 있습니다. (최대 5개 동시 등록 가능)
2. **두 가지 맞춤형 풀이 모드**
   - **실전 모드 (TEST)**: 실제 시험처럼 모든 문제를 푼 뒤 한 번에 채점하여 점수와 결과를 확인합니다.
   - **학습 모드 (LEARN)**: 한 문제를 풀 때마다 즉시 정답 및 해설을 확인하며 꼼꼼하게 학습할 수 있습니다.
3. **자동 오답 노트 시스템**
   - 실전 모드 또는 학습 모드에서 틀린 문제는 해당 시험의 '오답 노트'에 자동으로 누적됩니다.
   - 저장된 오답들만 모아서 **'오답 다시 풀기'** 기능을 통해 취약점을 보완할 수 있습니다.
4. **학습 데이터 로컬 저장**
   - 등록한 시험 목록, 오답 노트, 풀이 진행 상황이 브라우저의 로컬 스토리지(Local Storage)에 자동 저장되어 창을 닫아도 기록이 유지됩니다.
5. **데이터 내보내기**
   - 내가 푼 기록과 오답 노트 데이터를 JSON 파일 형태로 로컬 PC에 백업/내보내기 할 수 있습니다.

---

## JSON 데이터 작성 규칙 (JSON Schema)

본 CBT 시스템에 시험을 등록하기 위해서는 아래의 규칙에 맞게 작성된 JSON 데이터가 필요합니다. 파일은 **최상위 객체 안에 배열이 있는 형태**이거나 **배열 단독 형태** 모두 지원합니다.

### 1. 기본 구조
데이터는 `questions` 배열을 포함하는 객체 형태를 권장합니다.

```json
{
  "title": "무역영어 1급 기출문제",
  "questions": [
    {
      "id": 1,
      "question": "다음 중 밑줄 친 부분의 뜻으로 가장 알맞은 것은?\nWhat is <u>THIS</u>?",
      "options": [
        "선택지 1",
        "선택지 2",
        "선택지 3",
        "선택지 4"
      ],
      "answer": 0,
      "explanation": "THIS는 계약에서 ...를 의미합니다.\n\n**[보충 해설]**\n초보자도 쉽게 이해할 수 있는 풀이 요령..."
    }
  ]
}
```

### 2. 필드 설명 (Field Details)

- **`title`** *(선택 사항)* : 시험의 제목을 지정합니다. 생략 시 "사용자 등록 시험 1" 등으로 자동 부여됩니다.
- **`questions`** *(필수)* : 개별 문제들의 정보를 담고 있는 배열입니다.

`questions` 배열 내부의 각 문제 객체는 다음 필드를 반드시 포함해야 합니다:

| 필드명 | 타입 | 설명 |
| :--- | :--- | :--- |
| **`id`** | String / Number | 문제의 고유 식별자입니다. (오답 노트 관리 등에 쓰이므로 중복되지 않아야 합니다.) |
| **`question`** | String | 문제의 지문과 질문 텍스트입니다. <br>※ `\n`을 사용하여 줄바꿈을 할 수 있으며, `<u>텍스트</u>` 와 같은 HTML 태그를 삽입하면 밑줄 효과 등이 렌더링 됩니다. |
| **`options`** | Array | 문제의 객관식 보기들입니다. 문자열(String)로 구성된 배열이어야 합니다. |
| **`answer`** | Number | `options` 배열 내에서의 **정답 인덱스**입니다. (0부터 시작하므로, 첫 번째 보기가 정답이면 `0`을 입력합니다.) |
| **`explanation`** | String | 문제를 틀렸을 때나 학습 모드에서 표시될 해설 텍스트입니다. |

## 사용 방법 (How to Use)
1. `index.html` (또는 `CBT.html`) 파일을 더블클릭하여 웹 브라우저(Chrome, Edge 등)로 엽니다.
2. 우측 화면에서 **[PC에서 JSON 파일 첨부]** 버튼을 눌러 규격에 맞는 `.json` 파일을 업로드하거나 빈칸에 텍스트를 붙여넣습니다.
3. 하단의 **[시험 등록하기]** 버튼을 클릭합니다.
4. 등록된 시험 목록에서 원하는 시험을 선택하고 **학습 모드** 또는 **실전 모드**로 풀이를 시작합니다.
